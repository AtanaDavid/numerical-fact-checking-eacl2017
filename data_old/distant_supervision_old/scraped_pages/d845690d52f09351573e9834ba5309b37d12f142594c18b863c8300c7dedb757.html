Tennis Rules

Injuries and Retirements

We're making betting better by voiding your Match Result bet if the player you've backed retires injured after the first set - that's on top of paying out if you've backed the winner!

We offer a wide variety of markets on the sport of Tennis. From the Australian Open right through to the US Open - with Wimbledon inbetween! - choose from a huge choice of betting markets to enjoy.

General Rules

In the event of any of the following circumstances all bets will stand:

  • A change of schedule which affects the time or date of a match
  • A change of venue
  • A change from indoor court to outdoor court or vice versa
  • A change of surface (either before or during a match)

All Outright bets will be unaffected by any change of surface or venue.

Also:

  • In the event of any mid-match delays, bets will still stand.
  • Official Tournament results/statistics shall determine settlement for all markets.

Match Betting

You're betting on: who will win the match

  • If the first set is not completed, bets will be void. If the first set is completed but the game not played to its natural conclusion, bets will stand on the player deemed the winner (under tournament rules) while the opposing player will be made void.
  • If there is a change to the total number of sets to be played, bets to win the match will stand.

1st Set Betting

The rules below govern the following markets: 1st Set Winner, 1st Set Correct Score, 1st Set Tie Break, 1st Set Total Games

  • In the event of a disqualification or retirement all bets will be void unless the settlement of the bet has already been determined e.g. if a set is abandoned at 4-4, bets on Over/Under 9.5 games or fewer in the set are settled as winners/losers respectively, since any conclusion to the set would have to have had at least 10 games.

Games Betting

The rules below govern the following markets: Handicap Betting, Total Games, Total Games Live, Current Set Games, Next Set Games

  • In the event of a disqualification or retirement all bets will be void unless the settlement of the bet has already been determined e.g. if the match is abandoned at 6-4 4-4, bets on Over/Under 19.5 games or fewer in the match are settled as winners/losers respectively, since any conclusion to the match would have to have had at least 20 games.
  • In Doubles Matches where a Championship Tie Break is played in the final set, for betting purposes this will only count as 1 game.
  • If there is a change to the total number of sets to be played bets will be void.

  • In the event of ties, if no tie price is offered, bets will be void

Current Set Betting

The rules below govern the following markets: Current Set Winner, Current Set Score, Tie Break in Current Set, IPTL Set Winner, IPTL Set Winner (Doubles)

  • If the set is not completed, bets will be void. Alternatively:
  • Current Set Total Games: In the event of a disqualification or retirement all bets will be void unless the settlement of the bet has already been determined e.g. if a set is abandoned at 4-4, bets on Over/Under 9.5 games or fewer in the set are settled as winners/losers respectively, since any conclusion to the set would have to have had at least 10 games
  • Current Set Handicap Games: In the event of a disqualification or retirement all bets will be void unless the settlement of the bet has already been determined e.g. if a set is abandoned at 4-4, bets on a player +2.5 games in the set are settled as winners since any conclusion to the set will only be a difference of a maximum of 2 games.
  • Tie Break Correct Score: If a tie break doesn’t take place, bets will be void
  • IPTL Set Winner: Please note that sets in this competition are played first to 6 games but with a 4 minute 'shoot out' played if the set reaches 5-5. Set Winners in the same match cannot be combined within the same multiple.
  • IPTL Set Winner (Doubles): Bets will be settled on the team winning the set. Any substitutions made will not affect settlement.

Next Set Betting

The rules below govern the following markets: Next Set Winner, Next Set Correct Score, IPTL Set Winner, IPTL Set Winner (Doubles)

  • Next Set bets are singles only
  • If the set is not completed, bets will be void. Alternatively:
  • Next Set Total Games: In the event of a disqualification or retirement all bets will be void unless the settlement of the bet has already been determined e.g. if a set is abandoned at 4-4, bets on Over/Under 9.5 games or fewer in the set are settled as winners/losers respectively, since any conclusion to the set would have to have had at least 10 games
  • IPTL Set Winner: Please note that sets in this competition are played first to 6 games but with a 4 minute 'shoot out' played if the set reaches 5-5. Set Winners in the same match cannot be combined within the same multiple.
  • IPTL Set Winner (Doubles): Bets will be settled on the team winning the set. Any substitutions made will not affect settlement.

Next Game Betting

The player who is due to serve the game is denoted by (server). If the wrong player is indicated as (server) then any bets taken on that Next Game will be void, regardless of the result.

The rules below govern the following markets: Next Game Winner

  • Next Game bets are singles only
  • If any game is not completed, bets on that particular game will be void. If the umpire awards a penalty game, all bets on that game will be void.
  • In the event of a tie break, a separate market entitled ‘tie-break winner' may be offered. A tie break will not be covered under the next game market and all bets placed on that particular game will be void.

The rules below govern the following markets: Next Game Score

  • If any game is not completed, bets on that particular game will be void. If the umpire awards a penalty game, all bets on that game will be void.

The rules below govern the following markets: Next Game Receiving Points, Next Game 1st Point, Break Point in Next Game?, Deuce in Next Game?, Next Game Points, Ace in Next Game, Double Fault in Next Game

  • If any game is not completed, bets on that particular game will be void unless the settlement of the bet has already been determined. If the umpire awards a penalty game, all bets on that game will be void.
  • Next Game 1st Point - 1st Serve: The market will be settled on the first legal serve of the game. This means that if there is a let played or a point replayed for any purpose, the market will be settled on the completed point only.
